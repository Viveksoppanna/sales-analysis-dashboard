# Sales Analysis Dashboard (Power BI)

## 📊 Project Overview
Developed an interactive Power BI dashboard to analyze sales performance, revenue trends, and regional insights.

## 🚀 Features
- KPI tracking (Revenue, Profit, Quantity)
- Region-wise and category-wise analysis
- Interactive filters and slicers
- Drill-down capabilities

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
